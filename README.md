# Home-Data-Processing-and-Visualization-Toolkit

# Brief:

This Python project offers a comprehensive toolkit for managing, processing, and visualizing home financial data. The core function, load_data, reads data from CSV files, parses PDFs, and provides options to update CSVs and save data to SQLite.

Key Features:
-	Data Loading and Updating:
    -	Extracts credit card transactions and bank account data from PDF and Excel files, respectively.
    -	Allows in-place CSV updates and saving data to SQLite.
-	Data Preprocessing:
    -	Standardizes and classifies transaction descriptions.
    -	Identifies and deals with duplicate records.
-	Data Analysis and Visualization:
    -	Generates detailed analysis reports, including statistics, monthly and yearly totals.
    -	Provides interactive dashboards showcasing top vendors, channel breakdown, and yearly comparisons.
-	Data Validation:
    -	Offers comprehensive data validation tools to ensure data consistency and integrity.
    -	Visualizes and identifies differences between datasets and their distributions.
    
Overall, this toolkit streamlines the process of home financial data management, ensuring accuracy, consistency, and actionable insights through interactive visualizations.
